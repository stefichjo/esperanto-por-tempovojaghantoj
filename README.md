Zamenhof bazis ***Esperanton*** sur ***Speranto***, t.e. miksaĵo el akĉentoj de ***Lingvoo*** (**Linguo** *[lingŭo]*). ***Speranto*** prononciĝas precipe en la Slav-Ĝermana akĉento de ***Lingvoo***, kaj ĝi skribĝas en fonetika ortografio.
