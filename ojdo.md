## Gramatiko

### Substantivoj

Substantivoj finiĝas je **-o**. Ĝi estas ellasebla. Anstataŭ la multenombra finaĵo **-s** uziĝas **-i** post ellasita **-o**.

| |
|-|
| **domo** *[=]* |
| **dom'** *[=]* |
| |
| **domos** *[=]* |
| **domi** *[=]* |

Nomoj finiĝas je **-o** por vir-genro kaj **-a** por in-genro.

| Lingua | Oido |
|-|-|
| **Petrus** *[=]* | **Petro** *[=]* |
| **Petra** *[=]* | **Petra** *[=]* |

La vokativa finaĵo estas **-u**.
